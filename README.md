# python-flask-webapp

## Setup
```
  sudo apt install python3-pip -y
  sudo apt install python3-venv -y
  sudo apt install python3-flask -y

  python3 -m venv .
```


```
  source bin/activate
```

```
  pip3 install SQLAlchemy flask flask-jsonpify flask-sqlalchemy flask-restful
```

```
  FLASK_APP=src/app.py flask run
```


```
  http://localhost:5000/
```
